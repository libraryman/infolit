Several Different Types of Media

As we can see from the Library Catalog limiters, you can limit your search to books, eBooks, Thesis, Journals, DVDs and many more.

How do all these different types of media get into the Library Catalog?

Cataloging and Knowledge Base Modules

(i.e. Metadata)

Highly Trained Cataloging Librarians utilize the Metadata module to catalog books, print journals and physical media

For eBooks, electronic journals and articles Librarians select knowledge base collections that contain those resources

Physical Resources

  • 01. Purchase

    Libraries purchase print books and journals from several different places, some are even purchased through Amazon.

  • 02. Cataloging Print Resources

    (ORIGINAL CATALOGING)

    For material that has not been cataloged before, a highly trained librarian will create and input the data to create a bibliographic record then set the holdings

  • 03. Cataloging Print Resources

    (COPY CATALOGING)

    Depending on the type of physical resource being cataloged, one may use bibliographic records from resources already cataloged by other schools and then set the holdings or upload the record to the ILS

  • 04. Now available in the Public Library Catalog

    The material is now discoverable through the Library Catalog as well as WorldCat.org, allowing all libraries and universities to discover this material.

Electronic Resources

  • 01. Purchase

    Libraries purchase electronic journals and articles mainly from database vendors, who package and index the electronic journals and articles

  • 02. The Knowledge Base

    OCLC maintains a databases containing most of the electronic databases subscription packages

    Since the content of these databases constantly changes, by selecting the Knowledge Base holdings for a particular database, the holding will always be up to date

  • 04. Now available in the Public Library Catalog

    The material is now discoverable through the Library Catalog as well as WorldCat.org, allowing all libraries and universities to discover this material.

Now that all of these different types of media are in the Library Catalog, how do patrons check them out?

Patron Records

  • 01. Patron File

    The Library must obtain or compile are file with all of the enrolled student's information.

  • 02. Normalizing the Data

    Once the patron information is consolidated into one file, the data must be formatted according to the ILS's specifications

  • 03. Uploading the patron file

    The patron information is uploaded to the ILS, usually by FTP to the remote server

  • 04. Now you can check out books

    With the patron information loaded into the ILS, using the circulation module, Library staff are able to check out books and other library materials to students.

How does the Library Catalog know that the Book was checked out?





The Circulation Module connects to the Library Catalog and updates the holding location
